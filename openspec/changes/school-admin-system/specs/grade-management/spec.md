## ADDED Requirements

### Requirement: 教师录入成绩
教师 SHALL 能为自己负责的课程录入学生成绩，包含学生 ID、课程 ID、分数、学期。
分数 SHALL 为 0~100 之间的数值（可含一位小数）。
系统 SHALL 拒绝教师操作不属于自己的课程。
MVP 阶段同一学生同一课程同一学期 SHALL 只允许一条成绩记录（唯一约束）。

#### Scenario: 教师录入成绩成功
- **WHEN** 教师提交自己课程下学生的成绩，学期格式合法（如 "2024-1"）
- **THEN** 系统创建成绩记录，返回成绩 ID，HTTP 201

#### Scenario: 教师操作他人课程
- **WHEN** 教师提交不属于自己的课程 ID
- **THEN** 系统返回 HTTP 403，消息 "无权操作该课程成绩"

#### Scenario: 成绩超出范围
- **WHEN** 教师提交分数 < 0 或 > 100
- **THEN** 系统返回 HTTP 400，消息 "分数须在 0~100 之间"

#### Scenario: 重复录入同一成绩
- **WHEN** 同一学生同一课程同一学期已有成绩，教师再次录入
- **THEN** 系统返回 HTTP 400，消息 "该学生本学期此课程成绩已存在，请使用修改接口"

---

### Requirement: 教师修改成绩
教师 SHALL 能修改自己课程下已录入的学生成绩。

#### Scenario: 修改成绩成功
- **WHEN** 教师提交合法的成绩 ID 和新分数，且该成绩属于自己的课程
- **THEN** 系统更新分数，返回 HTTP 200

#### Scenario: 修改他人课程的成绩
- **WHEN** 教师提交不属于自己课程的成绩 ID
- **THEN** 系统返回 HTTP 403

---

### Requirement: 学生查询自己的成绩
学生 SHALL 能查询自己所有课程的成绩列表，支持按学期过滤。

#### Scenario: 学生查询成绩列表
- **WHEN** 已登录学生请求 `GET /api/grades/my`
- **THEN** 系统返回该学生所有成绩（含课程名称、学期、分数）

#### Scenario: 按学期过滤
- **WHEN** 学生传入 `semester=2024-1` 参数
- **THEN** 系统只返回该学期的成绩

#### Scenario: 学生查看他人成绩
- **WHEN** 学生尝试访问非本人的成绩
- **THEN** 系统返回 HTTP 403

---

### Requirement: 管理员查询成绩
管理员 SHALL 能查询任意学生的成绩，支持按学生 ID、课程 ID、学期过滤。

#### Scenario: 管理员查询指定学生全部成绩
- **WHEN** 管理员请求某学生 ID 的全部成绩
- **THEN** 系统返回该学生所有成绩记录，含课程名称和学期

#### Scenario: 管理员按课程查询全班成绩
- **WHEN** 管理员传入 `courseId` 和可选 `classId`
- **THEN** 系统返回该课程下（指定班级的）所有学生成绩列表

---

### Requirement: 教师查询本课程成绩
教师 SHALL 能查询自己课程下所有学生的成绩列表，支持按学期过滤。

#### Scenario: 教师查询课程成绩
- **WHEN** 教师请求 `GET /api/grades?courseId=<id>`，且该课程属于本人
- **THEN** 系统返回该课程所有学生的成绩（含学生姓名）
